This ML system contains the following functionalities

  1. Data Collection

This page takes the topics, subtopics and keywords as input from the user. When you hit submit, you are redirected to Models Status page

  1. Models Status

This page displays the processing status of all the models in the database

  1. Generate Summary

This page takes the required (processed) model name (fetched from the database) and zip file containing txts as input from the user. When you hit submit, you are redirected to Summaries status page. Here is a sample zip file that can be used to test this functionality: https://www.dropbox.com/s/h605bfbnfamsfd6/refined_ocred_data.zip?dl=0

  1. Summaries Status

This page displays the processing status of all the summaries in the database along with their respective model name. There is also an option to download and/or visualize the summary.
